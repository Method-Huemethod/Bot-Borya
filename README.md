# Bot Knowledge Base

Эта папка содержит подготовленные материалы для локального AI-бота.

## Структура
- `research/` — исследования в формате Markdown с метаданными.
- `shaman_transcripts/` — транскрипты шамана в формате Markdown с метаданными.
- `bot_instructions/` — инструкции для поведения бота.
- `index/` — индексы для поиска (`research_index.jsonl`, `transcripts_index.jsonl`, `knowledge_index.jsonl`, `source_manifest.csv`).

## Что сделано
- Исследования из `data/ayahuasca_corpus/parsed/*.json` приведены к единому текстовому формату.
- Транскрипты из `podcast/transcripts/*.txt` приведены к единому текстовому формату.
- К каждому документу добавлены метаданные: категория, название, источник, тема.
- Для поиска подготовлены JSONL-индексы с отдельными категориями источников.

## Принцип разделения источников
- `research` и `shaman_transcript` хранятся и индексируются раздельно.
- Общий индекс (`knowledge_index.jsonl`) сохраняет поле `category`, поэтому тип источника не теряется.
